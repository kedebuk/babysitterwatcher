Tanya Jawab

Punya pertanyaan?

Ini app apa sih sebenarnya?
+

Ellie adalah aplikasi monitoring bayi harian. Babysitter mencatat aktivitas si kecil (susu, makan, tidur, dll), dan orang tua bisa memantau semuanya real-time dari HP. Seperti punya "mata kedua" di rumah — tanpa harus WA bolak-balik.

Babysitter saya gaptek, bisa pakai?
+

Sangat bisa! Interface untuk babysitter dirancang sesimpel mungkin — tinggal pilih jenis aktivitas, isi detail, tap simpan. Kalau bisa pakai WhatsApp, pasti bisa pakai app ini. Banyak babysitter user kami yang langsung bisa tanpa diajari.

Beneran gratis? Ada biaya tersembunyi?
+

100% gratis selama early access. Tidak ada biaya tersembunyi, tidak perlu kartu kredit. Ke depan mungkin ada fitur premium tambahan, tapi semua fitur inti tetap gratis selamanya.

Data bayi saya aman nggak?
+

Sangat aman. Kami menggunakan Supabase dengan Row Level Security — artinya hanya kamu dan orang yang kamu undang yang bisa melihat data. Data dienkripsi dan disimpan di server aman. Tidak ada orang lain yang bisa mengakses data bayi kamu.

Bisa untuk lebih dari satu anak?
+

Bisa! Kamu bisa menambahkan beberapa anak dalam satu akun. Masing-masing punya dashboard dan riwayat terpisah, jadi data nggak tercampur.

Harus install app dari Play Store?
+

Tidak perlu! Ellie adalah web app — langsung buka dari browser HP. Bisa di-bookmark ke home screen supaya terasa seperti app native. Praktis, ringan, nggak makan storage.
